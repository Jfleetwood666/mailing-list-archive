Microsoft Office Support News Watch
===================================

As a subscriber to the Microsoft Office Support News Watch, you receive e-mail twice each month that highlights some of the new articles recently published on Support Online, Microsoft's award-winning technical support web site.


E-mail Security Alert
=====================

Recently Microsoft was notified by AUSCERT (http://www.auscert.org.au), OUSPG (http://www.oulu.fi) and NTBugtraq (http://ntbugtraq.ntadvice.com) of a security issue affecting the way Microsoft e-mail clients handle file attachments with extremely long file names.

On July 27th, Microsoft published a patch for Outlook 98 and Outlook Express 4.x that fixes the problems as reported. For more information about this problem and how to fix it, link to the following Microsoft security bulletin:

   http://www.microsoft.com/security/bulletins/ms98-008.htm

Microsoft has published the following two Knowledge Base (KB) articles on this issue:

   - Microsoft Knowledge Base (KB) article Q168019, Update Available For Outlook Express Security Issue,
http://support.microsoft.com/support/kb/articles/q168/0/19.asp

   - Microsoft Knowledge Base (KB) article Q175807, OL98:Update Available For Outlook 98 Security Issue,
http://support.microsoft.com/support/kb/articles/q175/8/07.asp

Get Support Online Articles by E-mail
=====================================

You can also receive these articles (and others) in e-mail by sending a message to mshelp@microsoft.com. In the Subject line of your message, enter the Article-ID number (Qnnnnnn). For example to receive Q162721, your Subject line should resemble the following example:

   Subject: Q162721

You can have multiple articles sent to you in e-mail by typing multiple Article-ID numbers separated by a comma. For example:

   Subject: Q178049, Q174914, Q174062

To receive an index of articles, enter "Index" (without quotation marks) in the Subject line. For example:

   Subject: Index

The MSHelp Index is updated monthly. For more information about MSHelp, see
http://support.microsoft.com/support/kb/articles/q183/1/21.asp.
