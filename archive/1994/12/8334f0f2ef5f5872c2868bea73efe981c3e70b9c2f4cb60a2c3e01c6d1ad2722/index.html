: 
: Last time I checked (a week ago, I am currently on the road) there

[..]

:               Because we can not support a 24 hour staff and we want

: scale up the trial. I promise you it will not be several more months 
: before everyone can join the trial.

There is a sort of (limited) Freenet system in Holland called DDS, 
people can request an account, (wich usually takes 48 hours), if youre
- -really- want ECash fast, get an Dutch-English dictionary and type:

    telnet dds.nl
