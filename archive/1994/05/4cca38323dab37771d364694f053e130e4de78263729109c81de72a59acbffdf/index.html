There has been a lot of speculation about the need to create new PGP 2.5
keys to keep on the mit keyserver.

I got a copy of 2.5 beta last night. I ran it, and created a new 1024-bit
key so Derek's work won't compromise my old 512 bit key. I then added my
old key to my 2.5 keyring.

Folks with long memories may recognize my key, I created it at NSA's
National Computer Security Conference in 1992.

Notice all the changes to make it 2.5 compliant.
